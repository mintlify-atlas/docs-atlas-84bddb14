Node to Code is an Unreal Engine editor plugin that converts Blueprint graphs into clean, structured C++ with a single click. Powered by leading LLM providers — including OpenAI, Anthropic Claude, Google Gemini, DeepSeek, Ollama, and LM Studio — it bridges the Blueprint-to-code gap for developers of every skill level.

## Get started in three steps

<Steps>
  <Step title="Install the plugin">
    Download the latest release from [GitHub Releases](https://github.com/protospatial/NodeToCode/releases) and install it to your Unreal Engine installation or project. See [Installation](/installation) for detailed instructions.
  </Step>
  <Step title="Configure your LLM provider">
    Open **Edit → Project Settings → Plugins → Node to Code**, select your preferred LLM provider, and enter your API key. For a local setup, configure Ollama or LM Studio instead. See [LLM Providers](/providers/overview).
  </Step>
  <Step title="Translate a Blueprint">
    Open any Blueprint in the Unreal editor, select the nodes you want to translate, then click the **Node to Code** button in the Blueprint Editor toolbar. Your C++ appears in the dockable output window with syntax highlighting and implementation notes.
  </Step>
</Steps>

<Note>
  Node to Code v1.2.2 supports Unreal Engine on **Windows (Win64)** and **macOS (Mac)**. Check [GitHub Releases](https://github.com/protospatial/NodeToCode/releases) for the latest build.
</Note>
